Interface Config

Schema for the application configuration.

Environment variables in CAPS_CASE are converted to these snakeCase properties. e.g. devMode is DEV_MODE in .env

Param: devMode

Enables the API documentation endpoint at /docs and allows running development scripts

Param: debugMode

Enables debug level logging and error responses

Param: devInitDb

Runs the database deletion and reinitialization function on startup

Param: port

The port to run the api server on; defaults to 3000

Param: oapiSpecFilePath

Generated field from application directory and tsoa.json
